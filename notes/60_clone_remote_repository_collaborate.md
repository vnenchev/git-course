# 60 Clone a remote repository

- Collaborators need access to the remote
    - A Github account
- Owner must grant access to private repositories
- Collaborators must clone repository - true for open sources projects
- On the project page you have several ways to clone repository
    - HTTPS / SSH / Github CLI
- `git clone <remote-url> <directory-name>`
- `git clone https://github.com/authorofrepo/repo_to_clone.git lynda_version`
- inside your current directory will be created lynda_version folder and content will be cloned
- if you omit the directory name it will use the repository name for creation
