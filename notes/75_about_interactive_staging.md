# 75 About interactive staging

- Allow staging portion of the selected files
- helps make smaller more focused commits
- it is an option of third party GUI tools but also built into Git itself
- two way to enter interactive mode:
    - `git add -i`
    - `git add --interactive`
- we will make a couple of changes, about.html and cart.html are edited
- instead of `git add about.html` we do `git add -i`
- you have numbers and letters to choose from
- help option gives you a short description of the commands
- quit or 7 command exits out of interactive mode
