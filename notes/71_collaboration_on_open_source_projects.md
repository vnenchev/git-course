# 71 Collaboration on open source projects

- when you work on private projects:
    - grant individuals read and write access
- open source projects
    - world has read access
    - project leaders have write access
- decide what changes do you want to make
- many projects include a README file
- review the existing projects issues and pull requests
- let others know the change you intend to make
- fork the project
- create a pull request
