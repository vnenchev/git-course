# 35 Branching overview

- branches are inexpensive and very easy to create
- can be viewed also with `git branch`
- develop features separate from other people work
- one working directory for all branches / the file version are swapped when switching between, your working directory is replaced with what is on the branch
- branches don't take a lot of resources, you can try new ideas, develop features separate from other work
- all branches are in the same repository
- git replaces file versions when branch changes
