# 30 Referencing commits

- SHA hash to reference, HEAD pointer references, BRANCH reference, TAG reference, ANCESTRY reference, you can use them all for reference
- `git log --oneline`
- when you have SHA you can reference => `git show 32klk` or `git show HEAD` or `git show main`
- all of the above commands in this case are references to the same commit
- you can reference ancestry of this pointers
- you can use references like these with many Git commands like `show, log, diff, revert`
